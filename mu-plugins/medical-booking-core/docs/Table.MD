# Custom Table của hệ thống

## Table list
- wp_travel_booking_booking_index: Lưu index Booking giúp lọc nhanh
- wp_travel_booking_booking: Lưu Booking
- wp_travel_booking_customer: Lưu thông tin custom của user
- wp_travel_booking_notifications: Lưu các thông báo, Telegram, SMTP Email
- wp_travel_booking_tour_scheduler

## DROP Table
```sql
DROP TABLE wp_travel_booking_booking_data,wp_travel_booking_booking_index, wp_travel_booking_notifications, wp_travel_booking_tour_scheduler
```
